# Thanh Nam FC
 A multi-page website for a football pitch rental business (named after Thành Nam, near Vũng Tàu, Vietnam). Built with Bootstrap 5. Visitors can view pitch rental info and pricing, browse a gear shop, read pitch rules, and get in touch.


## File Structure

webproject/
    home.html       Landing page with hero banner and "About Us" intro
    about.html      About the club
    fields.html     Pitch rental info
    rules.html      Pitch rules & booking/payment info
    shop.html       Gear shop
    contact.html    Contact form (mailto), social links, embedded Google Map
    style.css
    img/            banners, pitch/gear photos
